Skills 使用帮助

了解 MiniAgent 内置技能的使用场景、触发方式，以及如何扩展自定义技能

什么是 Skills

Skills 是 MiniAgent 的技能模块，每个 Skill 是一组专业的指令和工具，用于完成特定领域的任务。 当你发送消息时，Agent 会智能匹配相关技能并自动加载，让模型获得对应领域的专业能力。

如何使用 Skills

方式1 底部对话框左边

点击输入框左侧的 Skills 图标，在弹出面板中勾选需要的 Skill，然后输入你的请求。指定Skill是最精准的使用方式，但需要你对技能有一定的了解。

方式2 @ 调用

在输入框中使用 @技能名 显式指定要使用的 Skill。例如 @docx 帮我写一份Word报告。@shell-pptx 用 Shell 风格做一个用户操作手册PPT

方式3 自动匹配

直接在对话中输入自然语言内容，Agent 会根据关键词自动匹配相关 Skill。例如发送「帮我做一个PPT」会自动激活 PPT 相关技能。

工作文件夹与输入文件

使用技能处理文档时，需要将待处理的文件放入项目的工作文件夹中。

项目的工作文件夹位于项目根目录下的 output/ 目录。所有输入文件和生成的文件都统一存放在这个文件夹里。

操作步骤

  1. 放入待处理的文件

    在 MiniAgent 的项目根目录下找到 output/ 文件夹，将你需要处理的所有文件（如 .docx、.pdf、.pptx、图片等）放入该目录。

  2. 在对话中引用文件

    你可以直接告诉 Agent 要处理的文件名，或者点击输入框右侧的 回形针按钮 浏览选择 output/ 目录下的文件（浏览功能仅限本项目文件夹内的文件）。对于项目外的文件，请在文件管理器中复制完整路径，粘贴到回形针弹窗输入框后点击「插入」按钮即可。例如：「把 报告.docx 转成 PDF」

  3. 获取生成结果

    Agent 处理完成后，生成的文件也会输出到 output/ 文件夹中，你可以直接在该目录下找到处理后的文件。

docx
Word文档 .docx md2docx 报告 备忘录
创建、读取、编辑 Word 文档。支持目录、标题、页码、页眉等格式化功能，Markdown 转 Word，插入图片，处理修订标记与批注。
pdf
PDF .pdf文件
读取/提取 PDF 中的文本和表格，合并/拆分 PDF，旋转页面，添加水印，创建新 PDF，填写 PDF 表单，加密/解密，提取图片，OCR 识别。

创建/编辑 PowerPoint 演示文稿。支持模板、布局、母版、演讲者备注、批注，合并拆分幻灯片文件，以及 .pptx 文件的读取解析。
html-ppt
PPT 演示文稿 演讲稿 keynote slideshow 技术分享 小红书图文
HTML PPT 工作室：使用模板驱动方式创作精美的 HTML 演示文稿。支持多种风格主题、丰富布局和动画效果，输出为可键盘翻页的静态 HTML 文件。

演示文稿 PPT转网页 演讲 路演 slides
创建动画丰富的 HTML 演示文稿，支持从零创建或从 PowerPoint 文件转换。基于 beautiful-html-templates 模板库，适合需要丰富视觉效果的展示场景。
markitdown

通用文件格式转换工具，将 PDF、DOCX、PPTX、XLSX、图片（OCR）、音频（语音转文字）、HTML、CSV、JSON、XML、ZIP、EPub 等多种格式转换为 Markdown。

扩展自定义 Skill

MiniAgent 支持用户自定义 Skill，你可以根据自己的需求编写专业领域的技能模块。

Skill 存放路径

自定义 Skill 可以放在以下任意一个位置（按优先级排列）：
  1. 项目目录下的 skills 文件夹

    放置在 MiniAgent 项目根目录的 skills/ 目录下。这是项目的预制 Skills 目录，随项目一起分发，都经过测试能够正常使用。

  2. 用户home目录下(C:\Users\你的用户名\.miniagent)的 skills 文件夹

    放置在 ~/.miniagent/skills/ 目录下。这是用户个人专属的 Skills 目录，不会被项目更新影响。
